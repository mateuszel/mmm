# Claims and simulations

This is the language gate for the demo, pitch and repository.

## Real and source-backed

- Requirements summarized in [Official case](OFFICIAL_CASE.md).
- Public product models and timestamped store/marketplace pages listed in [Asset manifest](ASSET_MANIFEST.md).
- Arithmetic derived from the frozen values shown on screen.
- Von Halsky capabilities described on the official InPost page.

`Real listing` means only that the source page existed when captured. It does not validate authenticity, seller conduct, inventory or availability at playback time.

## Deterministic

- Scenario timing, message copy, price calculations, mandate rules and risk decisions.
- Search results selected for the scripted sequence.
- Evidence scoring, evaluation outcomes, notifications and final receipts.
- FX rate and delivery/protection values explicitly frozen for the demo.

## Simulated

- Search execution and retailer/marketplace integrations.
- Coupon testing and checkout state.
- AP2-aligned mandate representation and BLIK-style authorization.
- Every seller message, reply, evidence upload and negotiation outcome.
- Privacy relay, German call, transcripts, translation and waveform.
- Purchases, reservations, payments, receipts and continuing search.

## Required labels

- `Scripted demo · No live transactions`
- `Public reference offer · captured [date/time]`
- `Simulated seller conversation`
- `Simulated evidence`
- `Simulated seller verification call`
- `Simulated payment authorization`
- `AP2-aligned mandate concept`

## Forbidden claims

- Live adidas, eobuwie, PRM, Farfetch, OLX, eBay, InPost, Nintendo or Fujifilm integration.
- Real seller contact, real telephone relay, real BLIK payment or production Solidgate flow.
- Full AP2 compliance, guaranteed authenticity/fraud detection or guaranteed seller safety.
- Partnership, endorsement or data access that has not been confirmed.
- Any fictional message or risky conduct attributed to the real person behind a captured listing.

## Seller privacy rule

Use a real public listing only as product/context evidence. Redact all seller identity and contact information, replace the persona in scripted scenes, and state that the conversation is an alternate fictional branch. Never call, message or transact with the seller for the demo.
