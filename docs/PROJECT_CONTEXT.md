# Project context

**Status: Discovery**

**Case: Solidgate - AI Shopping Assistant**

**Team: Team MMM (three people)**

## What is confirmed

The official case asks for a demo-friendly AI deal-hunting agent that receives a plain-language shopping brief, monitors deterministic merchant events, matches the same product across messy listings, computes landed cost, filters bait/fake discounts, calibrates alerts, and prepares or simulates a purchase under standing consent and hard caps. The result must be explainable and measurable. PSP integration is not required.

Use [Official case](OFFICIAL_CASE.md) for the complete confirmed summary.

## What is provisional

The leading direction is a proactive shopping negotiator: it may check availability, analyze listing photos and condition, assess seller reviews/reputation, ask questions, negotiate within boundaries, and bring back an explainable proposal. A minimal brand page plus interactive demo is being considered. AP2, BLIK, outbound messaging, marketplaces, autonomous buying, hosting, and live integrations remain ideas, not commitments. See [Current concept](CURRENT_CONCEPT.md).

## What is open

The value proposition, core demo, primary differentiator, autonomy boundary, merchant fixtures, evaluation scope, model use, system boundaries, stack, deployment, and integration depth are unresolved. See [Open decisions](OPEN_DECISIONS.md).

## What must not happen yet

- No new application implementation, framework choice, dependencies, infrastructure, or API integrations.
- No claims of AP2 compliance, BLIK support, messenger/marketplace integration, autonomous payment, or production Solidgate integration.
- No permanent workstream assignments.
- No deletion of the earlier generic scaffold; it is preserved for team review.

## Existing repository content

Before the official case was captured, the repository received a generic application scaffold and vendored reference material. Those files were removed from the active tree to keep the shared workspace clean; they remain recoverable from Git history and an offline repository bundle. No product implementation is currently active.
