# Demo script

> **Status: Superseded generic scaffold script.** Do not treat this as the selected Solidgate demo. The final flow is an [open decision](OPEN_DECISIONS.md).

1. Open the generic control plane and point out DEMO MODE / no external calls.
2. Select the partner fixture relevant to the brief.
3. State the incident, goal, and business metric at risk.
4. Run the agent: context, read-only evidence, typed plan, and gated proposal.
5. Show the proposal rationale and impact estimate.
6. Approve the simulation and show the trace update; explicitly say no live partner action occurred.
7. Reset and replay if requested.

Fallback: use the generic scenario, which has no partner-specific dependency; if browser state misbehaves, run `pnpm demo:reset` and reload.
