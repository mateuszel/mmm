# Contributing

Use short-lived branches such as `feat/agent`, `feat/ui`, and `feat/demo`. Pull from `main` often, keep commits focused, never force-push shared branches, and merge only green changes. Copy `.env.example` to ignored `.env.local`; never share credentials in issues, logs, screenshots, or commits.

Before review: run `pnpm check`, reset and replay the demo, and update relevant documentation or decisions.
